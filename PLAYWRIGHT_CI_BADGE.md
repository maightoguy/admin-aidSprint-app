Playwright CI badge (add to your README)

Replace <OWNER> and <REPO> with your GitHub owner and repository name, and replace <WORKFLOW_FILE> if you renamed the workflow file.

Markdown badge example:

```markdown
[![Playwright E2E](https://github.com/<OWNER>/<REPO>/actions/workflows/playwright.yml/badge.svg)](https://github.com/<OWNER>/<REPO>/actions/workflows/playwright.yml)
```

HTML badge example:

```html
<a href="https://github.com/<OWNER>/<REPO>/actions/workflows/playwright.yml">
  <img src="https://github.com/<OWNER>/<REPO>/actions/workflows/playwright.yml/badge.svg" alt="Playwright E2E" />
</a>
```

If your workflow file is named differently, point the badge URL at that workflow path. For private repos the badge will only be visible to authenticated users with access.
